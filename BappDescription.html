This extension extends Burp to support variables, similar to Postman or Insomnia. Users can store and reuse values in Burp requests.

Features

  • Supports referencing values in requests from Repeater, Proxy, Intruder, Scanner, or Extension
  • Variables are referenced with the notation ((variableName))and can be included anywhere in the request
  • Data can be copied between projectds by using the import and export options, or using Burp's "Import project file" feature to import extension data (Pro only)

Usage

  1. Set the variable "name:value" pairs in the "Variables" tab
  2. Reference the variables in any request, using the notation ((variableName))
  3. Send the request
  4. Confirm the variable references were replaced by viewing the request using the "Logger" tool
